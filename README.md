### MuYun 低代码开发平台
#### 特点：
* 云原生
* 同步/异步双轨支持
* JVM系，第一梯队性能
* 分层次构建，按需插拔
* 程序员友好（口号，一切为了开发人员）
* 前后端分离
* 测试驱动开发
